# react-modale-jf

## Install

```bash
npm install --save react-modale-jf
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-modale-jf'
import 'react-modale-jf/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
